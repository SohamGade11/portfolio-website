Matplotlib & Seaborn Vizz

Two Hypotheses were made, proposing a relation between the GDP and waste generated/waste mismanaged per person a day.

  • More the GDP per Capita of an enity more would be the waste generation per person.
  • More the GDP per Capita of an entity more would be the waste mismanaged per person.
Plot 1

Conclusion

Generated waste tend to increase with entity's GDP per capita. Hence, there is enough evidence to support the claim.



Plot 2

Conclusion

Mismanaged waste does not increase with an increase in entity's GDP per capita. Hence, the evidence is enough to reject the claim.



POSTGRESQL

Creating main table in database using csv files.


DROP TABLE IF EXISTS plastic_waste CASCADE;

CREATE TABLE plastic_waste AS
SELECT rg.id, rg.entity, rg.code, rg.year, rg.waste_gen_perday,
  rm.waste_mis_perday, rg.gdp_percapita, rg.population, rg.region
FROM raw_data_gen rg
LEFT JOIN raw_data_mis rm ON rg.id = rm.id;
									

SELECT entity, code, year, waste_gen_perday AS wgpd_in_kg, waste_mis_perday AS wmpd_in_kg, gdp_percapita
 FROM plastic_waste WHERE (waste_gen_perday, waste_mis_perday) IS NOT NULL
 ORDER BY waste_mis_perday DESC LIMIT 5;
